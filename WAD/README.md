# Doom WAD Files

This folder contains the **DOOM1.WAD** shareware WAD file, freely redistributable per id Software's original terms.

## Included

- **DOOM1.WAD** — Doom shareware (Episode 1: Knee-Deep in the Dead)

## Installation

Copy the WAD file into your ErenshorDoom plugin folder:

```
<Erenshor>/BepInEx/plugins/ErenshorDoom/DOOM1.WAD
```

## Other WADs

ErenshorDoom also supports these WADs (not included — requires purchase or separate download):

- `DOOM.WAD` — Ultimate Doom (all 4 episodes)
- `DOOM2.WAD` — Doom II: Hell on Earth
- `FREEDOOM1.WAD` / `FREEDOOM2.WAD` — Free open-source alternatives from [Freedoom](https://freedoom.github.io/)
